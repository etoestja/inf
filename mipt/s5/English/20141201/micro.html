The problem

  • Do you know how does a washing machine work? Which part is missing?
    Answer: “the brain”. How to manage all the electronical stuff to make a device?
  • More complex than you think

Old solutions

  • Electromechanical control. ex. Power button which is a switch
  • Complex electrical schemes. ex. Channels on an old tv
  • Drawbacks: really complex, only basic functions
  • No “brain” at all

Microcontrollers

  • Lightweigh computer on a single chip: RAM, CPU, Flash
  • Usually no OS, just one program
  • Easy low-level I/O
  • Many different kinds. Typical Atmel AVR: 2K RAM, 20MHz CPU, 32K flash
  • Allow to easily control even complex devices

How to use it?

  • Write & compile on a PC, then upload to MCU.
  • Many languages: Assembler, C, C++, Pascal, ...
  • Pins are inputs (“voltmeter”, 0 or 1) or outputs (0 or 5V)
  • Easy to code: 
    
if(PINB & (1 << 0)) // reading from PB0, zero or one
{
	PORTB |= (1 << 5); // setting voltage on PB5, 0V or 5V
}

Applications

  • Consumer electronics (TV sets, washing machines, refrigerators, ...)
  • Industrial electronics (CNC)
  • Robots
  • Subsystems in big devices (ex. power management in a laptop)
  • ...
